# LogiAssist - Project Starter

This zip contains a starter project scaffold for **LogiAssist**:
- backend/ : Node.js + Express (RAG stubs, Supabase integration stubs)
- frontend/: Next.js minimal chat UI
- infra/: Supabase SQL to create tables
- docs/: demo script, pitch bullets, README

This is a working scaffold with TODOs. Replace API keys and implement production-grade vector queries and OCR pipeline.

See docs/demo-script.md for demo steps.
